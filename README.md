# coding-checker

 这是一个简单的对拍工具。
 
### 使用指南：

启动程序后会在当前目录生成`\coding-checker\`

点击`+`可以展开所有的对拍目录。

点击对应名称可进入控制界面。

`start`为启动对拍，在未全部完成编译前无法点击。

`open file`可以打开对拍错误样例所在目录。

`del file`可以删除先前所有对拍样例。

右侧点击`Edit ***`编辑对应源文件。
推荐在本地编辑器提前编辑完成后导入。

右侧点击`compile`编译对应源文件。
